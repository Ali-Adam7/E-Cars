# E-Cars
 Building E-Commerce Systems Project

How to run a microservice:

-For Development: <br>
 `npm run dev`

-For a Production build:
`npm run build` <br>
`npm run start` <br>


ORM Set up: <br>
`npm install @prisma/client` <br>
Sync database: <br>
`npx prisma migrate dev` <br>
Generate Prisma client: <br>
`npx prisma generate` <br>
